# gov-scan daily

한국 공공 공고 데일리 대시보드 — 공공기관 채용 · 주택청약 · 정부 R&D · 나라장터 입찰 · 학교장터 · 정책자금.

**페이지**: https://reasonofmoon.github.io/gov-scan-daily/

- 매일 아침 로컬 파이프라인이 8개 공공 소스(공공데이터포털 OpenAPI 2종 + 공개 공고 게시판 6종)를 수집해 직전 조사와 비교(diff)하고, 신규·마감 임박 공고를 정적 페이지로 게시합니다
- 목록 대시보드일 뿐 자격 판정이 아닙니다 — 마감·자격·조건은 수시로 바뀌므로 **신청 전 반드시 원문·접수기관에서 확인**하세요
- 데이터 출처: 조달청 나라장터·한국부동산원 청약홈(공공데이터포털 OpenAPI), IRIS·나라일터·잡알리오·S2B·중진공·소진공 공개 공고 페이지
- 수집기: [ir-search](https://github.com/djfksjd/ir-search)의 전수조사 패턴(전수 수집 → diff → 원문 링크 필수 · 불명 표기)을 공공 데이터 도메인으로 확장한 gov-scan 스킬
